# 📚 Estrutura de Dados - Repositório Pessoal

Este repositório reúne **anotações, tarefas, atividades práticas e projetos** desenvolvidos ao longo da disciplina de **Estrutura de Dados**, focando na implementação e compreensão de estruturas fundamentais como listas, pilhas, filas, árvores, entre outras.

Ressalto que a disciplina foi cursada no IDP durante a minha formação de Engenharia de Software. 

> A disciplina tem como objetivo **desenvolver o raciocínio lógico e a capacidade de organização de dados**, usando a linguagem C como base para as implementações.

Referência: [Repositório do Professor Jean Toso](https://github.com/jeanto/data_structure_course_notebook)
